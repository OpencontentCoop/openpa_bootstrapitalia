# OpenPA Bootstrap Italia

Estensione grafico-funzionale per Openpa conforme alle linee guida per i siti web della PA.

Riferimenti:
 - https://italia.github.io/bootstrap-italia/
 - https://italia.github.io/design-comuni-prototipi/
